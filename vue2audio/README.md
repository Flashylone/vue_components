# vue2audio
> 后台音频播放组件

## Installation
```shell
npm i vue2audio
```

## Usage
```javascript
import Vue from 'vue'
import vue2audio from 'vue2audio'

Vue.component('vue2audio', vue2audio)
```

### Attribute
| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| url | 必选参数, 音频的地址 | string | — | — |
